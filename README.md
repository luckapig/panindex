# PanIndex
[![](https://www.herokucdn.com/deploy/button.png)](https://heroku.com/deploy?template=https://github.com/libsgh/PanIndex-heroku.git)

## 简单的网盘目录列表
- [x] 跨平台、易部署
- [x] 多模式、多网盘
- [x] 多主题
- [x] 下载直链
- [x] 防盗链
- [x] 短链、分享
- [x] 访问控制

## 目前支持的网盘
- 本地目录
- 天翼云网盘
- Teambition（国内、国外）
- 阿里云盘
- OneDrive（国际版、世纪互联）
- 和彩云
- 谷歌云盘
- WebDav
- FTP
